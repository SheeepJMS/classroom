📋 使用方法

您可以通过以下方式在网站中使用这些图标：

1. 直接使用SVG文件

<img src="classroom_icon.svg" alt="课堂图标" width="64" height="64">

2. 作为CSS背景图片

.classroom-icon { background-image: url('classroom_icon.svg'); background-size: 64px 64px; width: 64px; height: 64px; }

3. 内联SVG代码

<svg width="64" height="64" viewBox="0 0 64 64"> <!-- SVG内容 --> </svg>

4. 在Flask模板中使用

<img src="{{ url_for('static', filename='icons/classroom_icon.svg') }}" alt="课堂图标">
